- **Highlights** — Synchronized audio, page-follow PDFs, deep-link cue copy

## Submission rules

- Site must be live and publicly reachable.
- Praetorius must be powering the works/repertoire surface (not just embedded in an iframe).
- Keep the canonical "Powered by Praetorius" attribution in the footer (see [Press Kit](/press/)).
- Provide a short blurb describing the stack, skin, and runtime used.

## Skins in the wild

The showcase tracks which skins are seeing real-world use. If you'd like to feature a skin we haven't shipped yet, open an issue tagged `skin:request` on the [GitHub repo](https://github.com/cbassuarez/praetorius).

- **Console** — terminal-flavored CLI surface for technical and academic contexts
- **Cards-Tabs** — gallery-style works board for portfolios and festival programs
- **Typefolio** — editorial typographic layouts for libraries and publishers
- **Vite-Breeze** — light, fast hero layout for landing pages and program notes
- **Kiosk** — full-screen presentation mode for galleries and lobby installs
- **Docs-Reader** — long-form reading mode for liner notes and program books
