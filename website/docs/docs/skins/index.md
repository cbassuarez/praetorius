# Skins

- **Typefolio** — editorial/portfolio; right-hand PDF preview.
- **Console** — terminal-style takeover; command palette, minimal chrome.
- **Vite-Breeze** — modern “liquid glass”; card grid + HUD.
- **Cards-Tabs** — neo-brutal gallery board + inspector tabs.
- **Kiosk** — touch-wall mode for installations and public display.
- **Docs-Reader** — long-form documentation with score/media modules.

Each skin renders the same data model and supports the global appearance system (`theme` mode + palette + cursor/effects where supported).

Try the stock two-work demo (Satie + Praetorius) in the [Playground](/playground), with live theme and palette switching.
