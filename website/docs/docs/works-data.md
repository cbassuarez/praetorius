# Works Data (Schema)  <SinceBadge since="v0.2.0" />

Define your works in a JSON that validates against the Praetorius schema.

<SchemaViewer src="/schema/works.schema.json" />

## Common Patterns
- Attach `pdf` and `audio` URLs per work.
- Provide optional `pageMap` for exact page-follow.
- Use stable `id` strings to enable deep links.

See also: `prae doctor` for validation.
